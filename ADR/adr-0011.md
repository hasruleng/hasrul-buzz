# ADR-0011: Shared visual identity — a cartographic "map of Hasrul's world"

← [ADR index](../ADR.md)

**Status:** Accepted (2026-06-21)
**Owed by:** [ADR-0005](adr-0005.md) ("shared visual identity becomes a real requirement … the *primary*
job of the Phase-2 work")
**Extends:** [ADR-0006](adr-0006.md) (the entry scene is where identity lands first) ·
[ADR-0004](adr-0004.md) (reskins the timeline as a route) · [ADR-0001](adr-0001.md) (content-first;
legibility is non-negotiable)
**Realised by / unifies:** [ADR-0008](adr-0008.md) (hub = map) · [ADR-0009](adr-0009.md) (vista =
"highest mountain") · [ADR-0010](adr-0010.md) (pace/route = the trail)

**Feeds:** [ROADMAP.md](../ROADMAP.md) Phase 2

## Context

[ADR-0005](adr-0005.md) made a single shared visual identity — palette, typography, motifs spanning the
readable site and the game — the *primary* Phase-2 deliverable, but did not choose one.
[ADR-0006](adr-0006.md) established *where* it first lands (the entry scene). The game ADRs since have
independently converged on spatial, journey-shaped language: a **hub world** of locations
([ADR-0008](adr-0008.md)), a panoramic **"highest mountain" overview** ([ADR-0009](adr-0009.md)), and a
**route/pace** as the one shown metric ([ADR-0010](adr-0010.md)). The readable site already renders the
career as a **vertical timeline** ([ADR-0004](adr-0004.md)) — which is a route by another name.

So the identity is not arbitrary taste: a **cartographic "map of Hasrul's world"** is the metaphor that
already fits every decision on the board. The career is a territory; stations are places; the journey is
a trail; discovery is exploration. One metaphor serves both modes — the readable timeline is a route on
the map, the game hub *is* the map.

The hard constraint, inherited from [ADR-0001](adr-0001.md): the audience includes time-pressed
recruiters. The aesthetic must read as **professional and stay fully legible** — it cannot tip into
skeuomorphic "olde map" clutter or illegible decorative type. Memorable, not gimmicky.

This also resolves an open thread: [ROADMAP.md](../ROADMAP.md) had gestured at a *retro pixel font /
8-bit* vibe (the hajj.buzz reference). That is **not** chosen — pixel-art reads as a gimmick to the
professional audience. The cartographic direction uses old-style serif + monospace instead (see
Decision); the roadmap line is updated in the same pass.

## Prior art

A visual-identity decision; references are map and editorial-design traditions.

**Antique explorer / nautical cartography** (parchment, compass rose, contour lines, route tracks,
a legend). *Borrowed:* the visual vocabulary — contours, station pins, a trail, a compass, a legend.
*Differs / explicitly not borrowed:* the skeuomorphic excess (faux-torn edges everywhere, sepia
overload, "ye olde" illegible lettering). We take the *language* of maps, kept clean and high-contrast.

**RPG world / overworld maps** (Zelda, Final Fantasy world maps; the climb-to-reveal-the-map beat).
*Borrowed:* the world-as-explorable-map that directly draws [ADR-0008](adr-0008.md)'s hub and
[ADR-0009](adr-0009.md)'s vista; locations linked by routes. *Not borrowed:* **fog-of-war** /
hidden-until-explored — it would gate content and break content-first ([ADR-0001](adr-0001.md)); nothing
on this map is hidden behind play.

**Cartographic data-viz & journey/route maps** (transit maps; Strava-style route + pace summaries).
*Borrowed:* a **route line as a data structure** — the career timeline rendered as a trail, which is
also the runner motif and the [ADR-0010](adr-0010.md) pace/route. *Differs:* the "distance" is symbolic.

**Editorial cartography** (National Geographic-grade maps: serious typography married to map
illustration). *Borrowed directly:* the proof that map aesthetics can be **professional, not childish** —
this is the answer to the recruiter-risk. Old-style serif labels + restrained color do the work.

**Internal prior art.** The existing readable timeline ([ADR-0004](adr-0004.md)) and the host avatar
([HasrulAvatar.tsx](../src/components/HasrulAvatar.tsx)) — whose charcoal/earth palette already sits
comfortably on parchment. Continuity, not a reset.

**What we did *not* borrow — and won't.** Pixel-art / 8-bit (rejected above); 3D / WebGL globe spectacle
(engineering-wow, against [ADR-0001](adr-0001.md)/[ADR-0005](adr-0005.md)); and any decorative choice
that lowers text contrast below accessible thresholds.

## Decision

Adopt a **cartographic "map of Hasrul's world"** as the one shared visual identity. Concretely:

**Palette — aged paper + ink + a single trail accent.** A warm parchment base, sepia/ink text, faded
contour lines, and the three persona accents recast as *map regions* (sea, ochre, moss). One distinct
**trail accent** (terracotta) is reserved for the route/journey line so the path always reads as the
through-line. (Exact hex values are tuned in the visual specimen accompanying this ADR; all text/background
pairings must meet WCAG AA.)

**Typography — old-style serif + monospace.** A serif for headings, content, and *map labels* (continues
Georgia / an old-style serif); a **monospace** for "coordinates," metadata, and the
[ADR-0010](adr-0010.md) pace/distance readout — so stats read like map coordinates. **No pixel font.**

**Motifs — the map vocabulary, used sparingly:**
- **The route/trail** — the unifying element. A dashed path linking station "pins"; a marker for "you
  are here." The readable timeline is rendered *as* this route; in the game it is the path between
  locations; under [ADR-0010](adr-0010.md) it draws itself as the avatar moves (the pace trail).
- **Topographic contour lines** — a subtle background texture, never competing with text.
- **Station pins, a compass rose, a legend** — pins mark stations; the legend doubles as the persona /
  lens key; the compass is a small recurring mark (favicon / loading / corner flourish).

**Apply it as a light reskin, not a rebuild** ([ADR-0005](adr-0005.md)). The change is the `:root`
palette + type tokens in [styles.css](../src/styles.css) and a route-styled timeline — the
[ADR-0003](adr-0003.md) content model and the Phase-1 router are untouched. The three-layer rule makes
this safe: this is a Render-layer skin.

**Legibility is an acceptance test, not an afterthought.** Recruiter-first ([ADR-0001](adr-0001.md)):
body text on parchment must stay at AA contrast; contour/texture sits well below text; the map vocabulary
must never cost a fast reader a single word.

**One identity, two modes.** The readable site gets the calm, editorial-map treatment; the game uses the
*same* tokens at higher saturation/animation. They must read as one product (the [ADR-0005](adr-0005.md)
requirement) — same parchment, same trail accent, same pins, same compass.

## Alternatives considered

- **Editorial + subtle game accents** (keep today's clean type; add a thin route line + motion) — safest,
  lowest effort, but the *least* memorable and it leaves the game and readable site only loosely related.
  *Rejected — under-delivers on [ADR-0005](adr-0005.md)'s "one identity, memorable" goal; though its
  restraint is folded into this ADR's legibility-first rule.*
- **Terminal / dev aesthetic** (monospace, dark code-editor vibe) — distinctive and on-theme for an
  engineer, but leans *hacker* over *product*, fits the game poorly (a map world isn't a terminal), and
  dark-mode-only narrows the editorial feel. *Rejected — wrong fit for the journey/hub metaphor.*
- **Retro pixel / 8-bit arcade** (the [ROADMAP.md](../ROADMAP.md) gesture) — maximal game-feel, but reads
  as a gimmick to recruiters/founders and fights the product-judgment north star. *Rejected — and the
  roadmap's pixel-font line is retired with it.*
- **Full skeuomorphic antique map** (heavy textures, torn edges, sepia everything) — atmospheric but
  hurts legibility and looks costume-y. *Rejected — we take the map *language*, not the costume.*

## Consequences

- **A light reskin of the readable site is owed in Phase 2** — `:root` palette + type tokens in
  [styles.css](../src/styles.css); the [ADR-0004](adr-0004.md) timeline restyled as a route with pins.
  No content-model or router change ([ADR-0003](adr-0003.md) intact).
- **The game inherits a ready-made visual vocabulary** — hub = map ([ADR-0008](adr-0008.md)), vista =
  the map's high ground ([ADR-0009](adr-0009.md)), route/pace = the trail ([ADR-0010](adr-0010.md)).
  The identity decision and the game-structure decisions now reinforce each other.
- **A small set of flat-SVG motif assets is owed** — contour background, compass rose, station pin /
  "you are here" marker, trail/route line. Cheap, engine-free, reusable in both modes.
- **The pixel-font direction is dropped**; [ROADMAP.md](../ROADMAP.md) Phase 2 is updated to say
  cartographic (serif + monospace), not retro-pixel.
- **Accessibility is a hard gate** — AA contrast on parchment is part of "done" for the reskin.
- **A visual specimen accompanies this ADR** (palette swatches, type specimen, motifs applied to a
  sample timeline + hub) so the concrete execution is reviewed before the reskin ships.
- **Supersede trigger:** if the cartographic reskin measurably hurts reach-the-content rate or reads as
  gimmicky in feedback, a future ADR may fall back to the editorial + subtle-accents alternative —
  recorded as a new ADR, never an in-place edit here.
